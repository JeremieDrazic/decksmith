Effect Legend

primary Diagonal gradient (lighter center → base → darker edge) + translateY(-1px) + amber/violet outer glow + shimmer sweep on entry
secondary Border brightens + bottom radial glow grows inward from below + translateY(-1px) + no shimmer (cleaner for outlined)
ghost Radial gradient bg fades in from center (ambient glow) + text color brightens — no translate, stays flat intentionally
destructive Ember gradient (bright center → base → deep) + intense red outer glow + translateY(-1px) + shimmer sweep
active translateY(+1px) (press-in) + shadow collapses + inset shadow for depth — opposite of hover
loading Label visibility:hidden (width preserved) + centered spinner + aria-busy="true" + pointer-events:none
